## How to use

1. Copy the `docs` folder into the root of `isaiahmurray-sudo/resin-poll-data-collector`.
2. The two uploaded screenshots are already included as `docs/assets/gui-overview.png` and `docs/assets/live-graphs.png`.
3. The current uploaded recording is included as `docs/assets/draft-demo-recording.mp4`, but it is only a short draft clip. Replace it with the final 4 to 5 minute video or link to the longer hosted version before submitting.
4. Commit and push.
5. In GitHub, go to Settings → Pages.
6. Set Source to `Deploy from a branch`.
7. Set Branch to `main` and Folder to `/docs`.
8. Save.

Your Pages URL should look like:

`https://isaiahmurray-sudo.github.io/resin-poll-data-collector/`
